# NVC Feelings Inventory

True feelings in NVC describe internal emotional states, not judgments about what others did to us.

## When Needs Are Met

### Affectionate
compassionate, fond, loving, openhearted, tender, warm

### Engaged
absorbed, curious, engrossed, enchanted, fascinated, interested

### Excited
amazed, ardent, aroused, dazzled, energetic, enlivened, enthusiastic, exuberant

### Exhilarated
radiant, electrified, euphoric, overjoyed, thrilled

### Grateful
appreciative, moved, thankful, touched

### Happy
amused, blissful, cheerful, delighted, ecstatic, elated, giddy, glad, jolly, joyful

### Hopeful
confident, expectant, lighthearted, optimistic, upbeat

### Inspired
eager, enthused, motivated, psyched, stimulated, stirred

### Peaceful
calm, comfortable, centered, content, fulfilled, mellow, quiet, relaxed, serene

### Refreshed
recharged, rejuvenated, renewed, rested, restored, revived

## When Needs Are Not Met

### Anger
aggravated, angry, annoyed, disgruntled, enraged, exasperated, furious, irritated

### Aversion
appalled, bothered, disgusted, horrified

### Confusion
ambivalent, baffled, bewildered, conflicted, perplexed, puzzled

### Disconnection
apathetic, bored, detached, distant, indifferent, numb

### Disquiet
agitated, alarmed, concerned, dismayed, disturbed, frustrated, restless, upset

### Embarrassment
ashamed, flustered, mortified, self-conscious

### Fatigue
burnt out, depleted, exhausted, tired, weary

### Fear
afraid, anxious, apprehensive, frightened, insecure, panicked, scared, terrified, worried

### Pain
aching, anguished, devastated, grief, heartbroken, hurting, lonely, miserable

### Sadness
depressed, dejected, despairing, disappointed, discouraged, disheartened, hopeless

### Tension
distressed, edgy, frazzled, jittery, nervous, overwhelmed, stressed

### Yearning
longing, nostalgic, pining

## Faux Feelings (Not True Feelings)

These words imply what someone else did to you rather than describing your internal state. When you notice these, look for the true feeling underneath.

| Faux Feeling | Possible True Feeling |
|--------------|----------------------|
| abandoned | scared, lonely, sad |
| attacked | scared, angry |
| betrayed | hurt, angry, disappointed |
| blamed | angry, scared, confused |
| bullied | scared, angry, hurt |

| coerced | angry, scared, resentful |
| diminished | hurt, sad, angry |
| ignored | lonely, hurt, sad |
| insulted | angry, embarrassed, hurt |
| intimidated | scared, anxious |
| manipulated | angry, scared, resentful |
| misunderstood | frustrated, sad, lonely |
| neglected | lonely, sad, hurt |
| pressured | anxious, overwhelmed, resentful |

| taken for granted | sad, hurt, angry |

| unheard | frustrated, sad, lonely |
| unsupported | sad, lonely, scared |
| used | angry, hurt, sad |
